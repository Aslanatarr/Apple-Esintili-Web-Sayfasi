
Apple Store Arayüz Çalışması 🍎
Apple'ın modern ve minimalist tasarım dilini temel alan, HTML5, CSS3 ve Vanilla JavaScript kullanılarak geliştirilmiş dinamik bir web projesidir.

✨ Özellikler
Dinamik Sepet Sistemi: Ürün ekleme, çıkarma ve miktar yönetimi.

LocalStorage Entegrasyonu: Tarayıcı kapatılsa bile sepet verileri korunur.

Dinamik Ödeme Sayfası: Sepet verilerini otomatik çeken profesyonel checkout ekranı.

Modern Tasarım: CSS Grid ve Flexbox ile oluşturulmuş Apple stili arayüz.

İnteraktif SSS: Kullanıcı dostu akordeon (details/summary) soru-cevap bölümü.

İletişim Formu: Şık ve etkileşimli kullanıcı geri bildirim alanı.

🛠 Kullanılan Teknolojiler
HTML5 - Sayfa yapısı ve semantik etiketler.

CSS3 - Özelleştirilmiş Apple stili arayüz ve responsive tasarım.

JavaScript (ES6+) - Sepet mantığı, DOM manipülasyonu ve LocalStorage yönetimi.

📂 Dosya Yapısı
Plaintext
├── index.html          # Ana mağaza sayfası
├── odeme.html         # Sepet özeti ve ödeme ekranı
└── Images/            # Ürün görselleri (AirPods, iPhone, Mac vb.)
